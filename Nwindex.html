Projects

Bootstrap

This project involved building a responsive e-commerce website using the Bootstrap framework. I utilized Bootstrap's grid system to create a dynamic product listing page that adapts seamlessly to different screen sizes. Initially, I struggled with understanding how Bootstrap's grid worked, but through practice and by consulting the documentation, I was able to master it. I also gained experience in customizing Bootstrap's default styling using CSS overrides to match the client's branding guidelines. Integrating Bootstrap's form components with my backend logic for handling user registrations was another valuable learning experience. This project reinforced the importance of mobile-first design, as Bootstrap's responsive framework made it easy to prioritize the mobile experience. Link to project

CSS

This project involved building a responsive website for a local bakery using HTML and CSS. I focused on creating a visually appealing design that showcases their delicious products. I learned how to use flexbox to create a flexible and adaptable layout for the homepage, and I implemented media queries to ensure the website looks great on all devices. One challenge I faced was creating a visually appealing image gallery, but I overcame this by learning about CSS grid and its ability to create dynamic layouts.
